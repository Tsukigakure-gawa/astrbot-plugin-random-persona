# 🎭 随机人格 (Random Persona)

AstrBot 插件 — 随机化 AI 情感基调和表达方式，让回复不再千篇一律。

## 解决的问题

AI 聊天机器人在长期对话中容易表现出三个"非人"特征：

1. **永远正面积极** — 不会低落、不会敷衍、不会不耐烦
2. **永远有回应** — 哪怕不需要回复也要硬找话题
3. **永远一种腔调** — 缺少人类自然切换表达方式的多样性

本插件通过动态情感状态 + 表达方式随机化 + 沉默权，让 AI 的回复呈现更接近人类的自然波动。

## 核心机制

### 1. 四维情感状态

每次会话随机初始化，随对话自然漂移：

| 维度 | 含义 | 影响 |
|------|------|------|
| 能量 (energy) | 唤醒度 | 低→慵懒简短 / 高→兴奋活跃 |
| 情绪 (valence) | 效价 | 低→冷淡疏离 / 高→热情亲切 |
| 耐心 (patience) | 持续下降 | 低→敷衍 / 高→详细展开 |
| 开放性 (openness) | 主动程度 | 低→不问不答 / 高→主动延伸话题 |

### 2. 五种表达方式

加权随机 + 情感状态影响权重：

- **记叙** — 讲述经历、描述过程
- **说明** — 客观陈述，条理清楚
- **描写** — 注重细节，具象化呈现
- **议论** — 给出分析、判断和观点
- **抒情** — 表达情绪感受，有人情味

### 3. 沉默权

AI 有权在不需要回复时发送极简回应：

- 用户没问问题 + 上轮 AI 已充分回复 → 沉默
- 耐心极低 (< 0.2) → 沉默
- 低能 + 偏冷 → 30% 概率沉默

### 4. 随机扰动

- 回复长度 ± 变化
- 10% 概率切换表达方式
- 15% 概率注入语气词 / 口头禅

## 安装

将插件文件夹放入 AstrBot 的 `data/plugins/` 目录即可。

## 命令

| 命令 | 功能 |
|------|------|
| `/persona` | 查看当前人格状态 |
| `/persona random` | 随机重置所有维度 |
| `/persona chill` | 低能偏冷模式 |
| `/persona warm` | 高能偏暖模式 |
| `/persona talkative` | 话多模式 |
| `/persona quiet` | 少说模式 |
| `/persona mode <模式>` | 切换表达方式 |
| `/persona off` | 关闭随机人格 |
| `/persona on` | 重新开启 |

## 配置

在 AstrBot WebUI → 插件管理 → 随机人格 → 配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 沉默模式 | 触发沉默权时的行为 | 短回应 |

## 设计灵感和原则

基于 Anthropic 2026 年 "Emotion Concepts and their Function in a Large Language Model" 论文（171 个情绪向量因果驱动行为），本插件通过 prompt 层模拟情绪波动 + 表达方式随机化实现。

设计原则：
- **不假装有意识** — 情绪是风格模拟，不是声称主观体验
- **核心能力不降级** — 实质问题时准确性优先于风格
- **适度不可预测** — 模拟自然的不一致性，不牺牲可用性
- **敏感话题不嬉皮** — 严肃话题自动锁定高耐心 + 平实说明
